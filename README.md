# Ruby Simple Boilerplate

## On macOS & Linux

### Install & Run

```bash
./run
```

## On Windows

### Requirements

* Ruby 1.9+ (https://rubyinstaller.org/)
* Bundler (https://bundler.io/)

### Install & Run

```
bundle install
ruby script.rb
```
